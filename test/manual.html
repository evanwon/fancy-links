🔗 Fancy Links Extension Test Page

Use this page to test your Firefox extension functionality

📋 Testing Instructions

Test the extension by:

  1. Toolbar Button: Click the Fancy Links icon in the Firefox toolbar to open the popup
  2. Keyboard Shortcut: Press Ctrl+Alt+C to copy using your default format
  3. Settings: Right-click the extension icon and select "Preferences" to access settings

🎯 Expected Formats

When you copy this page, you should get different formats like:

  • Slack: <file:///path/to/test.html|Test Page for Fancy Links Extension>
  • Discord: [Test Page for Fancy Links Extension](file:///path/to/test.html)
  • Markdown: [Test Page for Fancy Links Extension](file:///path/to/test.html)
  • HTML: <a href="file:///path/to/test.html">Test Page for Fancy Links Extension</a>
  • Plain Text: Test Page for Fancy Links Extension - file:///path/to/test.html

🧪 Test Cases

This page tests:

  • ✅ Normal page title extraction
  • ✅ Local file URL handling
  • ✅ Title with special characters: "Test Page for Fancy Links Extension"
  • ✅ Long title handling (this title is moderate length)

📝 Manual Test Checklist

  • ☐ Extension icon appears in toolbar
  • ☐ Clicking icon opens popup with current page info
  • ☐ Settings gear icon appears in popup header (visible without scrolling)
  • ☐ Settings gear icon is clickable and opens settings page
  • ☐ All 6 format buttons work in popup
  • ☐ Keyboard shortcut Ctrl+Alt+C works
  • ☐ Settings page opens and saves preferences
  • ☐ Default format selection works
  • ☐ Copied text appears correctly in clipboard
  • ☐ Notifications appear when copying
  • ☐ Tab navigation includes settings button (accessibility)
  • ☐ Settings button has proper hover/focus states

🌐 Additional Testing

For comprehensive testing, also try the extension on:

  • A real website (like GitHub)
  • A page with no title
  • A page with a very long title
  • A page with special characters in the title
